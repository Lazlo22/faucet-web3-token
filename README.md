### Build using Vite

**First run** `npm install`
<br/>
<br/>
`npm run dev` &ndash; **starts locally a server**
<br/>
<br/>
`npm run build` &ndash; **builds the project**
<br/>
<br/>
`npm run preview` &ndash; **Vite preview function (previews locally a build version)**
